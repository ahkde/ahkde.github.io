TrayTip

Erstellt in der Nähe des Tray-Icons eine Sprechblase. Benötigt Windows 2000/XP oder höher.

TrayTip [, Titel, Text, Sekunden, Optionen]

Parameter

Titel

Wenn kein Parameter angegeben ist, dann werden alle aktuell angezeigten TrayTip-Fenster entfernt.

Andernfalls enthält dieser Parameter den Fenstertitel, der bis zu 73 Zeichen lang sein kann (alle darüberliegenden Zeichen werden nicht dargestellt).

Wenn der Titel leer ist, dann wird die Titelzeile der Sprechblase komplett weggelassen und kürzt das Fenster vertikal.
Text

Wenn dieser Parameter leer oder nicht vorhanden ist, dann werden alle aktuell angezeigten TrayTip-Fenster entfernt.

Andernfalls kann eine Nachricht angegeben werden, die unter dem Titel angezeigt wird. Nur die ersten 265 Zeichen des Textes werden angezeigt. CR-Zeichen (`r) oder LF-Zeichen (` n) können verwendet werden, um mehrere Textzeilen zu erstellen. Zum Beispiel: Zeile1`nZeile2

Wenn Text lang ist, dann kann er in mehreren kurzen Zeilen aufgeteilt werden (Fortsetzungsbereich), die die Lesbarkeit und Wartbarkeit verbessern könnten.
Sekunden

Die ungefähre Anzahl an Sekunden, um das Fenster anzuzeigen, dass danach automatisch vom Betriebssystem entfernt wird. Wird eine Zahl kleiner als 10 oder größer als 30 verwendet, dann tritt das Minimum (10) oder Maximum (30) in Effekt. Wenn leer oder nicht vorhanden, dann wird für gewöhnlich die minimale Zeit verwendet. Dieser Parameter kann ein Ausdruck sein.

Die tatsächliche Zeitüberschreitung weicht vom Angegebenen ab. Microsoft: "Wenn der Benutzer den Computer scheinbar nicht verwendet, dann zählt das System diese Zeit nicht für die Zeitüberschreitung." (Technische Details hier) Deshalb kann für die genaue Kontrolle, wie lange das TrayTip-Fenster angezeigt wird, der Sleep-Befehl gefolgt vom TrayTip-Befehl ohne Parameter verwendet werden, oder auch SetTimer wie im Beispielabschnitt.
Optionen

Verwendet eine der folgenden Ziffern, um ein kleines Icon auf der linken Seite des Titels anzuzeigen:

1: Info-Icon
2: Warnung-Icon
3: Fehler-Icon

Wenn weggelassen, standardmäßig 0 (kein Icon). Dieser Parameter kann ein Ausdruck sein.

Bemerkungen

TrayTip benötigt Windows 2000/XP oder höher. In Windows 9x/NT funktioniert der Befehl nicht.

Die Sprechblase kann nicht angezeigt werden, wenn das Script kein Tray-Icon hat (durch #NoTrayIcon oder Menu, Tray, NoIcon). Wenn der folgende REG_DWORD-Wert vorhanden und auf 0 gesetzt ist, funktioniert TrayTip auch nicht:
HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced >> EnableBalloonTips

Das Betriebssystem spielt normalerweise einen Sound beim Anzeigen einer Sprechblase ab. Dieser Sound kann deaktiviert werden, indem eine 16 mit dem Optionen-Parameter addiert wird (z. B. 16 ist kein Sound und kein Icon, 17 ist kein Sound aber mit Info-Icon, usw.)

Zusätzlich wird ein ToolTip angezeigt, wenn der Benutzer den Mauszeiger über das Tray-Icon des Scripts hält. Der Inhalt des Tooltips kann wie folgt geändert werden: Menu, Tray, Tip, Neuer Text

Verwandte Befehle

ToolTip, SetTimer, Menu, SplashTextOn, MsgBox, InputBox, FileSelectFile, FileSelectFolder

Beispiele

TrayTip, Eigener Titel, Mehrzeiliger`nText, 20, 17

; Um genauere Kontrolle über die Anzeigezeit zu haben,
; ohne dabei Sleep zu verwenden (weil Sleep den aktuellen Thread stoppt):
#Persistent
TrayTip, Zeitlich festgelegter TrayTip, Wird für 5 Sekunden angezeigt.
SetTimer, TrayTipEntfernen, 5000
Return

TrayTipEntfernen:
SetTimer, TrayTipEntfernen, Off
TrayTip
Return

; Damit ein TrayTip dauerhaft angezeigt wird, verwendet einen Timer, um es regelmäßig zu aktualisieren:
SetTimer, TrayTipAktualisieren, 1000
Gosub, TrayTipAktualisieren ; Einmal aufrufen, um es sofort zu starten.
Return
