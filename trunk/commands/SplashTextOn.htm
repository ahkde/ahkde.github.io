SplashTextOn / SplashTextOff

Erstellt ein anpassbares Popup-Fenster.

SplashTextOff
SplashTextOn [, Breite, Höhe, Titel, Text]

Parameter

Breite Die Breite des Fensters (in Pixel). 200 ist Standard. Dieser Parameter kann ein Ausdruck sein.
Höhe Die Höhe des Fensters (ohne die Titelleiste, wenn das Script die AUT-Dateierweiterung nicht hat). 0 ist Standard (bedeutet, dass nur die Titelleiste angezeigt). Dieser Parameter kann ein Ausdruck sein.
Titel Der Titel des Fensters. Standardmäßig leer.
Text Der Text des Fensters. Standardmäßig leer. Wenn Text lang ist, dann kann er in mehreren kurzen Zeilen aufgeteilt werden (Fortsetzungsbereich), die die Lesbarkeit und Wartbarkeit verbessern könnten.

Bemerkungen

Für mehr Kontrolle über Layout und Schriftart/-farbe/-größe verwendet den Progress-Befehl mit der "zh0"-Option, um die Titelleiste wegzulassen und nur den Text anzuzeigen. Zum Beispiel: Progress, zh0 fs18, Beliebiger Anzeigetext mit Schriftgröße 18.

Verwendet den SplashTextOff-Befehl, um ein vorhandenes Splash-Fenster zu entfernen.

Das Splash-Fenster bleibt "immer im Vordergrund". Um dies zu ändern, verwendet WinSet, AlwaysOnTop, Off, . Mit WinSet kann das Splash-Fenster auch transparent gemacht werden.

Mit WinMove kann die Position und Größe des Splash-Fensters neu angepasst werden, wenn es angezeigt wird.

Im Gegensatz zu Progress, SplashImage, MsgBox, InputBox, FileSelectFile und FileSelectFolder ist nur ein Splash-Fenster pro Script möglich.

Wenn SplashTextOn verwendet wird, während das Splash-Fenster bereits angezeigt wird, dann wird dieses Fenster mit neuen Parametern ersetzt. Anstatt allerdings jedesmal das Splash-Fenster zum Ändern des Titels oder Textes neu zu erstellen, können für eine bessere Performance folgende Beispiele verwendet werden, besonders wenn das Fenster häufig geändert werden muss.

Verwandte Befehle

Progress, SplashImage, ToolTip, MsgBox, InputBox, FileSelectFile, FileSelectFolder, WinMove, WinSet
