FileGetSize

Ermittelt die Größe einer Datei.

FileGetSize, Ausgabevariable [, Dateiname, Einheit]

Parameter

Ausgabevariable Der Name der Variable, in der die ermittelte Größe gespeichert wird (auf die nächste ganze Zahl abgerundet).
Dateiname Der Name der Datei, die sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. Wenn nicht vorhanden, dann wird die aktuelle Datei in der innersten dateilesende Schleife verwendet.
Einheit

Falls vorhanden, bewirkt dieser Parameter, dass das Ergebnis in einer bestimmten Einheit ausgegeben wird (Bytes ist Standard):
K = Kilobytes
M = Megabytes

ErrorLevel

Bei Problemen wird ErrorLevel auf 1 gesetzt, ansonsten enthält ErrorLevel eine 0.

Die interne Variable A_LastError enthält den Rückgabewert der Systemfunktion GetLastError().

Bemerkungen

Dateien beliebiger Größe werden unterstützt, selbst wenn sie über 4 GB groß sind, und selbst wenn die Einheit in Bytes ist.

Wenn Dateiname ein Verzeichnis ist, dann wird eine Größe abhängig vom Betriebssystem ausgegeben (in den meisten Fällen wahrscheinlich 0).

Um die Größe eines Ordners zu berechnen, siehe folgendes Beispiel:

SetBatchLines, -1  ; Führt die Operation mit maximaler Geschwindigkeit aus.
Ordnergröße = 0
FileSelectFolder, BeliebigerOrdner  ; Fordert den Benutzer auf, einen Ordner auszuwählen.
Loop, %BeliebigerOrdner%\*.*, , 1
    Ordnergröße += %A_LoopFileSize%
MsgBox Der Ordner %BeliebigerOrdner% hat eine Größe von %Ordnergröße% Bytes.

Verwandte Befehle

FileGetAttrib, FileSetAttrib, FileGetTime, FileSetTime, FileGetVersion, dateilesende Schleife

Beispiel

FileGetSize, Ausgabevariable, C:\Eigene Dokumente\test.doc  ; Ermittelt die Größe in Bytes.
FileGetSize, Ausgabevariable, C:\Eigene Dokumente\test.doc, K  ; Ermittelt die Größe in Kilobytes.
