AutoHotkey deinstallieren

Nachfolgend finden Sie Anweisungen, wie Sie AutoHotkey auf Ihrem Betriebssystem deinstallieren können. Je nachdem, welche Version Sie haben, können sich die Schritte etwas unterscheiden.
Bitte navigieren Sie zu der Anweisung, die zu Ihrem Betriebssystem passt.

  • Windows 10
  • Windows 8, 7 oder Vista
  • Windows XP

Windows 10

  1. Schließen Sie alle AutoHotkey-Skripte und -Prozesse auf Ihrem Computer.
  2. Klicken Sie auf das Startmenü und dann auf Einstellungen.
  3. Klicken Sie auf Apps.
  4. Suchen und klicken Sie unter "Apps & Features" auf AutoHotkey.
  5. Klicken Sie auf Deinstallieren.
  6. Bestätigen Sie das mit einem Klick auf Deinstallieren.
  7. Klicken Sie auf Deinstallieren.

Windows 8, 7 oder Vista

  1. Schließen Sie alle AutoHotkey-Skripte und -Prozesse auf Ihrem Computer.
  2. Öffnen Sie die Systemsteuerung:
    • Windows 7 & Vista: Klicken Sie auf das Startmenü und dann auf Systemsteuerung.
    • Windows 8: Bewegen Sie den Mauszeiger in die rechte obere Ecke des Bildschirms. Klicken Sie auf Einstellungen und dann auf Systemsteuerung.
  3. Klicken Sie auf "Programm deinstallieren" oder "Programme und Features".
  4. Doppelklicken Sie auf AutoHotkey.
  5. Klicken Sie auf Deinstallieren.

Windows XP

  1. Schließen Sie alle AutoHotkey-Skripte und -Prozesse auf Ihrem Computer.
  2. Klicken Sie auf das Startmenü und dann auf Systemsteuerung.
  3. Klicken Sie auf Software.
  4. Klicken Sie auf AutoHotkey.
  5. Klicken Sie auf Entfernen.
  6. Klicken Sie auf Deinstallieren.
