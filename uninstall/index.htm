AutoHotkey deinstallieren

Nachfolgend findet ihr Anweisungen, wie ihr AutoHotkey auf eurem Betriebssystem deinstallieren könnt. Je nachdem, welche Version ihr habt, können sich die Schritte etwas unterscheiden.
Bitte navigiert zu den Anweisungen eures Betriebssystems.

  • Windows 10
  • Windows 8, 7 oder Vista
  • Windows XP

Windows 10

  1. Schließt alle AutoHotkey-Skripte und -Prozesse auf eurem Computer.
  2. Klickt auf das Startmenü und dann auf Einstellungen.
  3. Klickt auf Apps.
  4. Sucht und klickt unter "Apps & Features" auf AutoHotkey.
  5. Klickt auf Deinstallieren.
  6. Bestätigt das mit einem Klick auf Deinstallieren.
  7. Klickt auf Deinstallieren.

Windows 8, 7 oder Vista

  1. Schließt alle AutoHotkey-Skripte und -Prozesse auf eurem Computer.
  2. Öffnet die Systemsteuerung:
    • Windows 7 & Vista: Klickt auf das Startmenü und dann auf Systemsteuerung.
    • Windows 8: Bewegt den Mauszeiger in die rechte obere Ecke des Bildschirms. Klickt auf Einstellungen und dann auf Systemsteuerung.
  3. Klickt auf "Programm deinstallieren" oder "Programme und Features".
  4. Doppelklickt auf AutoHotkey.
  5. Klickt auf Deinstallieren.

Windows XP

  1. Schließt alle AutoHotkey-Skripte und -Prozesse auf eurem Computer.
  2. Klickt auf das Startmenü und dann auf Systemsteuerung.
  3. Klickt auf Software.
  4. Klickt auf AutoHotkey.
  5. Klickt auf Entfernen.
  6. Klickt auf Deinstallieren.
