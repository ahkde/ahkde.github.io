JavaScript

Die Dokumentation nutzt JavaScript, eine in Webbrowsern integrierte Skriptsprache, um interaktive Features zu ermöglichen, die in reinem HTML nicht verfügbar sind. JavaScript ist in allen gängigen Browsern standardmäßig aktiviert. Wir empfehlen, es aktiviert zu lassen, um die Nutzung der Dokumentation zu verbessern.

Die Dokumentation stellt Features bereit, die stark von den verfügbaren Features einer klassischen CHM-Datei inspiriert sind:

  • Randleiste mit Elementen wie:
    • Reiter "Inhalt", um einfach durch die Dokumentation zu navigieren
    • Reiter "Index", um anhand gängiger Stichwörter und Kategorien Dokumente zu öffnen
    • Reiter "Suche", um eine Textsuche mit optionaler Hervorhebung der Suchbegriffe durchzuführen
    • Liste von Überschriften im aktuellen Dokument
  • Toolleiste mit Tools wie:
    • Sprache wechseln
    • Version wechseln
    • aktuelles Dokument auf GitHub bearbeiten
    • zurück/vorwärts navigieren
    • Text größer/kleiner machen
    • aktuelles Dokument drucken
    • aktuelles Dokument im Standardbrowser öffnen
    • dunkles/helles Farbschema umschalten
    • Starteinstellungen der Hilfe ändern
  • Codeboxverbesserungen wie Syntaxhervorhebung und das Herunterladen von Code
