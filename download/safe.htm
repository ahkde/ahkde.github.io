Safe Browsing

Es ist bekannt, dass Google Safe Browsing, ein von Google Chrome und anderen Webbrowsern genutzter Dienst, angeblich "schädliche Programme" auf unseren Download-Seiten gefunden hat. So eine Warnmeldung könnte wie folgt aussehen:

Die Website, die Sie aufrufen möchten, enthält schädliche Programme.

Unbefugte Dritte könnten auf autohotkey.com versuchen, Sie zur Installation von Programmen zu bewegen, die sich nachteilig auf Ihre Browsernutzung auswirken. Dabei kann zum Beispiel Ihre Startseite geändert werden oder es erscheinen zusätzliche Anzeigen auf von Ihnen besuchten Websites.

Das oben Genannte entspricht nicht unseren Absichten. Der Verdacht liegt nahe, dass solche Warnseiten hauptsächlich durch Antiviren-Falschmeldungen oder undurchsichtige Richtlinien seitens Google in Bezug auf die Klassifizierung unerwünschter Software ausgelöst werden.

Diskussion und weitere Informationen zu diesem Thema finden Sie in diversen Forumsbeiträgen wie Google is blocking autohotkey.com.

Wenn Sie vorhaben, AutoHotkey wieder zu deinstallieren, aber nicht wissen, wie man am besten vorgeht, können Sie sich die detaillierten Anweisungen anschauen.

Antiviren-Falschmeldungen

Siehe dazu die häufig gestellte Frage: Mein Antivirus-Programm hat AutoHotkey oder ein kompiliertes Skript als Malware erkannt. Ist das wirklich ein Virus?

Falschmeldungen (False-Positives) treten oft auf, wenn eine neue Version veröffentlicht wird, da jede Version einen neuen Satz ausführbarer Dateien enthält, die die Antiviren-Anbieter noch nie zuvor gesehen haben. Die Bekämpfung solcher Falschmeldungen ist ein Kampf gegen Windmühlen, da man sie jedes Mal, wenn sie auftreten, bei jedem Antivirenanbieter einreichen muss. Jeder Anbieter hat sein eigenes Verfahren zum Annehmen und Bearbeiten solcher Einreichungen.

Wenn Sie als AutoHotkey-Benutzer auf so eine Falschmeldung stoßen, ist es am besten, die EXE-Datei zur Analyse an Ihren Antivirenanbieter zu senden. Der Anbieter kann dann bestätigen, ob die Datei tatsächlich infiziert ist und seine Virendatenbank entsprechend anpassen.
